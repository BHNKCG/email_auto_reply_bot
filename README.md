# Automated Email Reply Bot 
Beginner-friendly AI Automation project built using Python.

## Features
- Detects email intent using keywords
- Generates automatic email replies
- Clean and modular Python structure

## Project Structure
```
email_auto_reply_bot/
     ├── main.py
     ├── keyword_rules.py
     ├── reply_templates.py
     ├── requirements.txt
     ├── .gitignore
     └── README.md
```
## How to Run
1. Clone the repository:

       git clone https://github.com/BHNKCG/email_auto_reply_bot.git

2. Go to the project folder:

       cd email_auto_reply_bot

3. Run the bot:

       python main.py
